📖 Systemintegration Fachglossar

Zusammenfassung wichtiger Kernbegriffe moderner Unternehmensinfrastrukturen im Rahmen des Selbststudiums.

Active Directory (AD)

Der zentrale Verzeichnisdienst von Microsoft Windows Server Systemen. Es ermöglicht Administratoren, die gesamte Organisation von Benutzern, Computern, Berechtigungen und Gruppenrichtlinien an einem einzigen, zentralen Punkt sicher zu verwalten.

Virtualisierung (Hyper-V / Proxmox)

Das Aufteilen physischer Hardware-Ressourcen eines einzelnen Servers in mehrere voneinander isolierte, virtuelle Maschinen (VMs). Dies sorgt für maximale Hardware-Auslastung, enorme Stromersparnis und schnelle Datensicherungen.

RAID-Systeme (Redundant Array of Independent Disks)

Der Zusammenschluss mehrerer physischer Festplatten zu einem logischen Laufwerk. **RAID 1** spiegelt Daten für Ausfallsicherheit, während **RAID 5** Performance und Datenredundanz durch Paritätsdaten kombiniert.

3-2-1 Backup-Regel

Ein essenzielles Datensicherungskonzept: Es sollten mindestens **3 Kopien** der Daten existieren, auf **2 unterschiedlichen Medien** (z.B. NAS und externe Festplatte), wobei **1 Kopie komplett außerhalb** des Betriebs (z. B. Cloud-Backup oder Schließfach) gelagert wird.
